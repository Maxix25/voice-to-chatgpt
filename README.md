# voice-to-chatgpt

# What is this?
- It's a way to use ChatGPT using only your voice

# How can I run it?
- First, clone the repository using ```git clone https://github.com/Maxix25/voice-to-chatgpt.git```
- Then cd into the folder that's created using ```cd voice-to-chatgpt```
- Install the requirements using ```pip3 install -r requirements.txt```
- Set your <a href="https://platform.openai.com/account/api-keys">OpenAI API Key</a> in the .env file
- Run the code using ```python3 main.py```
- Now you can speak into your system's default microphone and get the ChatGPT response
